Menu Sequence and Description
The Description will be displayed as the menu item.

The Sequence Number allows you to sequence the menu items in your Menu component. The e-Business Framework Wizard automatically assigns any new menu item to the end of the Menu component.

You can control the sequence by assigning a different sequence number to the item. For example, to make the item appear at the top of the menu, use a Sequence Number of 1.

The Sequence Numbers are automatically assigned using intervals of 1.
