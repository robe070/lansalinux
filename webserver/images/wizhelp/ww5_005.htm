Select Menu Component
Once you select the Menu characteristic and press Go on the Wizard Home Page you will be presented with the Define Left Menu screen.

The Left Menu tab is automatically pre-selected. To define a Right or Horizontal Menu definition click on the appropriate tab. The steps to customize a Left, Right or Horizontal Menu are identical.

The Menu Features tab, on the right side of the screen, allows you to set up default values to be used when defining and building your Left, Right and Horizontal Menus. For more details, refer to Menu Features

To customize the system default menus, select the process DEFAULT on the left hand menu of the Wizard. The steps in customizing the system default menus or the menus for a specific process are the same.
