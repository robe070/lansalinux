Images
The company logo image will be used in your process layouts if you have selected to include the company logo in your layout using the Layout Features.

The image for the LANSA messages is often displayed beside the LANSA messages. The Wizard message interface has been modified not to include the messages image. Refer to Other Default Pages in the LANSA for the Web Guide for further information.

There are no process specific variables to override these values.

You must enter a value for both images if they are used in your Web Application. If a value is not entered, your Web pages will fail to locate and load the file.
