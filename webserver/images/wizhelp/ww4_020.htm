Tips & Techniques
  • When a color graphic variable is set to blank, LANSA for the Web will ignore this value and look for another default. A blank color is treated as if the variable does not exist.
  • To set the background image back to the system default, do not blank out the image file name. You must manually delete the *LW3PBGI_<process name> graphic variable using the LANSA for the Web Editor.
  • If you use a Wizard Supported Color for the Left/Horizontal Menu Color in combination with selecting layout schema one, three or five it is highly recommended that the background color be set to white (#FFFFFF) for the best results.
  • If you change the Left/Horizontal Menu Color from a Wizard Supported Color to a manually entered color, or vice-versa, the layout should be re-built to remove any references to the graphics used as the corner images. This only applies if you have selected layout schema one, three or five.
