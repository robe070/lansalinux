Wizard does not show the most current layout definition
While executing the Wizard multiple times, you may not see the most current information. When you access a screen, it shows the old data. Your last changes are not shown.

While executing the Wizard, your latest changes seem to be ignored and the old settings are being used instead.

Solution

It is very important that you have your browser properly configured when using the e-Business Framework Wizard. You do not want to be using cached images when you customize your application as your browser will display old settings. You should set your browser to retrieve the most current page each time a page is accessed using the Wizard.

If you are using an Internet Explorer browser (recommended), be sure to edit your Internet Settings for Temporary Internet files as described in Browser Configuration.

That is, your browser must check for newer versions of stored pages: Every visit to the page.

Document in cache must be compared to document on network: Every time.
