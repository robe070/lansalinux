New LANSA for Web Installation
If this is your first LANSA for the Web system, the e-Business Framework Wizard can be applied to all your new processes and functions provided you have installed the Wizard in the partition.

Refer to the Installing LANSA on iSeries Guide or the Installing LANSA on Windows Guide for how to install the e-Business Framework Wizard either into a new LANSA installation or a new partition.

If you import an application created from a LANSA system that did not have the e-Business Framework Wizard installed, refer to Existing LANSA for Web Installation.
