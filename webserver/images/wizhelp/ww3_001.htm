Wizard Home Page
When you execute the e-Business Framework Wizard (refer to Running the e-Business Framework Wizard) you will be presented with the Wizard Home Page as your starting point. The Wizard Home Page is the main menu for the Wizard where you return to in order to modify your selections.

The following actions can be initiated from the Wizard Home Page:

Home

Define Process Group

File Search

Language

Characteristics to Modify

Process Selection

Go
