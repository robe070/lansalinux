Click Tracking does not log events
You have elected to include a Click Tracking using the Layout Features then built your layout, but there are no details in your trace files (DC@W26 and DC@W27) after running your Web application.

Solution

Create a component <process name>_TRACE or DEFAULT_TRACE with the appropriate TRACE_DETAIL or TRACE_SUMMARY statement embedded in the page.

Use the Editor and check the correct LAYOUT page exists, <process name>_LAYOUT or DEFAULT_LAYOUT. Review the HTML and look for the tag <RDML TRACE> near the top of the page. If this tag does not appear, then you have not included the component. Use the Wizard to include the missing Layout Feature then rebuild your layout.

For more details, refer to Enabling Event Logging in the LANSA for the Web Guide.
