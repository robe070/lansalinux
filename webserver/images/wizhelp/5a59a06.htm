Style Sheets
If your menu includes dynamic buttons, the generated menu component will include references to two associated Cascading Style Sheet classes. The classes used are as follows:
MenuClasses
Left Menu.leftmenu

.selleftmenu
Right Menu.rightmenu

.selrightmenu
Horizontal Menu.hrzmenu

.selhrzmenu


A default style sheet, which includes the above classes, will be generated when you build a layout (if a style sheet does not already exist for the current process). The default style sheet removes the underline typically associated with a hyperlink and sets the default text color and weight. For more details, refer to Customizing a Presentation Layout – Technically Speaking.
