Company Logo does not appear in the presentation layout
You have elected to include the Company Logo using the Layout Features then built your layout, but your function does not show a company logo when it executes.

Solution

You must define the Company Logo by modifying the appropriate System Variable in the Wizard. Ensure an image file has been entered and that the file exists.

Use the Editor and check the correct LAYOUT page exists, <process name>_LAYOUT or DEFAULT_LAYOUT. Review the HTML and look for the tag <RDML SETTMPFLD="14" VALUE="1"> near the top of the page. If this tag does not appear or the value is not set to "1", then you have not included the component. Use the Wizard to include the missing Layout Feature then rebuild your layout.
