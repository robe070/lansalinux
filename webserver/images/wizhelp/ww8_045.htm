Tips & Techniques
  • Remember that the size of the images used will impact the performance of your Web applications.
  • If you wish to see a list of the image files available, use File Search to review the files stored in the image location specified on the Web server.
  • Once you have specified an image file, the Wizard creates an image file graphic variable. To reset this field, you must manually delete the graphic variable using the LANSA for the Web Editor. Do not set the field to blank characters otherwise your application will receive a file not found error. Your application will search for a blank file name.
