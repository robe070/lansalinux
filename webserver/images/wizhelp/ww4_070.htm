Layout Schemas
The e-Business Framework Wizard is shipped with five layout schemas, each defined by a component (WIZLAYO01 through WIZLAYO05). They are installed into your partition when you import the e-Business Framework Wizard (save file LWEBWIZ2).
Do NOT modify the pages associated with these components or use this naming convention to create your own layouts as future releases of the e-Business Framework Wizard may update these pages.
When you have completed the customization of the presentation layout, the e-Business Framework Wizard creates a new layout page:
 <process name>_LAYOUT
where <process name> is the name of your current LANSA process (this will be DEFAULT for the system default component)

The layout pages are built from the layout schema components shipped with the e-Business Framework Wizard. The resulting layout page is then used when you run your LANSA application. A comment tag in your HTML page will indicate which of the five layouts was selected.
